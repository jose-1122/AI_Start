Predict Future Sales¶

* 대회 링크 : https://www.kaggle.com/c/competitive-data-science-predict-future-sales/¶

  • 대회 목표 :

    러시아 최대 소프트웨어 회사 중 하나인 1C Company에서 제공하는 일일 판매 데이터(2013년 1월 ~ 2015년 10월)를 바탕으로, 2015년 11월 모든 shop(매장)의 item(제품) 총 판매량 예측.

  • 대회 평가 지표 : root mean squared error (RMSE).

[ 파일 설명 ]¶

  • sales_train.csv : 훈련 세트. 2013년 1월부터 2015년 10월까지의 일일 기록 데이터.
  • test.csv : 테스트 세트. 2015년 11월 shop과 item의 매출을 예측해야 함.
  • sample_submission.csv : 제출 파일.
  • items.csv : item에 대한 추가 정보.
  • item_categories.csv : item 카테고리에 대한 추가 정보.
  • shops.csv : shop에 대한 추가 정보.

[ 데이터 설명 ]¶

  • date : dd / mm / yyyy 형식의 날짜.
  • date_block_num : 편의를 위해 사용되는 연속된 월 번호. ex) 2013년 1월은 0, 2013년 2월은 1, ... 2015년 10월은 33.
  • shop_id : shop의 고유 식별자.
  • item_id : item의 고유 식별자.
  • item_price : item 가격.
  • item_cnt_day : 판매된 item 개수.
  • ID :테스트 세트 내에서 (shop, item) 튜플을 나타내는 ID.
  • item_name : item 이름.
  • shop_name : shop 이름.
  • item_category_id : item 카테고리의 고유 식별자.
  • item_category_name : item 카테고리 이름.
